# XOA Script Documentation - Packet Capture

## Introduction
What this script example does:
1. Connect to a tester
2. Reserve a port
3. Configure capture trigger criteria
4. Start capture
5. Wait a while
6. Stop capture
7. Check capture status
8. Read captures packets from the buffer and show them one by one
9. Disconnect from the chassis
